# Code of Conduct

## Our standard

Be respectful, constructive, and welcoming. Discuss ideas and code without harassment, discrimination, personal attacks, or disclosure of private information.

## Scope and enforcement

This standard applies in repository issues, pull requests, discussions, and other project spaces. Report conduct concerns privately through the repository's security-advisory channel. Maintainers may edit or remove contributions and restrict participation when behavior is harmful.

Enforcement decisions should consider context, severity, and the goal of restoring a safe collaborative environment.
