# Security Policy

## Supported versions

Security fixes currently target the latest code on `main` and the latest published release.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's private vulnerability reporting or security advisory feature for this repository:

https://github.com/JoanOliver04/media-batch-converter/security/advisories/new

Include the affected version, reproduction steps, impact, and any suggested mitigation. Do not include unrelated personal data or private media. Maintainers will acknowledge reports when available and coordinate disclosure after a fix or mitigation is ready.

This policy does not promise a particular response time or provide a security guarantee.
